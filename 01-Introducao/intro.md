# Introdução à Engenharia de Software

### O que é a Disciplina?
* **Entender o problema** antes de programar.
* Modelar soluções
* Comunicar ideias técnicas.

### Por que é importante?
* Erros custam caro
* Software mal feito gera retrabalho

### O que é Software
**NÃO é só código e programação.**
* Código-fonte
* Documentação
* Testes
* Manutenção

---

## Engenharia de Software
Aplicação de princípios de engenharia
no desenvolvimento de software.
* Qualidade
* Organização
* Previsibilidade
* Manutenção

### Programar X Engenharia de Software

* **Programar:** escrever código.
* **Engenharia:** planejar, projetar, construir, testar e manter.

---

## Processos de Software
Conjunto de atividades organizadas para desenvolver software.
Define o que fazer, quando fazer, quem faz e como fazer.

### Etapas Comuns

* Levantamento de requisitos
* Análise
* Projeto
* Implementação
* Testes
* Manutenção

#### Modelos
* Cascata
* Incremental
