### Objetivo:
Fornecer uma visão geral dos processos de desenvolvimento de software.
Apresentar a linguagem de modelagem UML (Unified Modeling Language) e seus
diagramas de definição, com foco em modelos de casos de uso e modelo de
classes. Especificar projetos de software orientado a objetos. Apresentar técnicas
para a elicitação de requisitos e para a escrita de documentação de modelagem
de software.

### Ementa:
- Visão Geral da Engenharia de Software
- Análise e Especificação de Requisitos
- Processos de Desenvolvimento de Software (tradicionais e ágeis)
- A linguagem de modelagem unificada (UML)
- Casos de Uso
- Modelagem de Classes
- Cenários de Casos de Uso
- Técnicas de documentação de modelagem de software
