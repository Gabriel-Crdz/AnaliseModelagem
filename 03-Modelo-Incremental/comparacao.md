# Cascata X Incremental

| **Critério** | **Modelo Cascata** | **Modelo Incremental** |
|----------|---------|-----------|
| Estrutura |Sequencial e linear | Cíclico e evolutivo |
| Ordem das fases | Uma fase termina para começar outra | Pequenos ciclos repetidos | 
| Entrega | Apenas no final | Entregas parciais |
| Validação de requisitos | No início do projeto | Validação contínua |
| Flexibilidade a mudanças | Baixa | Alta |
| Risco | Alto(erros aparecem no final) | Reduzido(erros detectados cedo) |
| Feedback cliente | Tardio | Contínuo |
| Evolução | Dificil após inicio | Natural e esperada |
| Arquitetura | Planejada no inicio | Evolui ao longo do projeto |
| Indicado para | Requisitos estaveis | Projeto com incerteza ou evolução |
